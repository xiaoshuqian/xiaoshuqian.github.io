Detailed outline

  • Data, variables, operators
    • Scoping rules:
      • Read: LEGB (Local->Enclosed->Global->Built-in)
      • Write: declare using non-local/global. If you don't declare using non-local/global, it will be treated as a local and will cause an exception if the variable has already been used as a non-local.
    • Mutable vs Immutable, id() and type()
      • Mutable: The value can be changed without change the id(). Including list, dict, set, byte array
      • Immutable: If the value is changed, the id() has to change. Include string, tuple, frozenset
    • Chained comparison/assignment: a < x > b is equivalent to a < x and x > b
    • Bitwise: bit shift >>, <<; if (m & 0x1) or (n & 0x1): img[m,n] = 255
  • Branching
  • Loops and comprehension
  • Function
  • Class
  • Standard modules
    • Built-in functions (68 in python 3.6):
      • ord, chr
      • id, type
      • input, print, format, open, close
      • int, float, complex, str, repr, hex, oct, bin, bool, iter, next
      • eval, exec
      • map, filter
      • round, pow, round, divmod, min, max, abs, sum, any, all, sorted, reversed, len
      • set, dict, list, frozenset, tuple, bytearray, enumerate, zip
      • locals, globals
      • isinstance, issubclass, vars, getattr, hasattri, delattr
    • Collections: namedtuple, deque, Counter, Defaultdict, OrderedDict
    • queue: SimpleQueue (3.7), Queue, PriorityQueue, LifoQueue
    • unittest
      • import unittest
      • class TestMethod(unittest.TestCase): def test1(self): self.assertEqual(actual, expected);
      • unittest.main()
    • functools
      • sorted(xs, key=functools.cmp_to_key(cmp_fun))
      • functools.reduce(function, iterable[, initializer])
      • @functools.total_ordering
      • functools.partial(func, *args, **keywords)
    • itertools
      • for item in chain(xs, ys): cycle(xs): accumulate(xs): accumulate(xs, mult):
      • product('ABC', repeat=2): AA AB AC BA BB BC CA CB CC
      • permutations('ABCD', 2): AB AC AD BA BC BD CA CB CD DA DB DC
      • combinations('ABCD', 2): AB AC AD BC BD CD
      • combinations_with_replacement('ABCD', 2): AA AB AC AD BB BC BD CC CD DD
  • Numerical
  • numpy: linalg.solve()
  • scipy: optimize.linprog()
