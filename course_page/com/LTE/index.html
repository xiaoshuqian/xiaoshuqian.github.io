Description

Detailed outline

  • Digital front end, Tx and Rx
    • Single component carrier case: freq down conversion, down-sampling
      • IQ and DC estimation and compensation
      • AGC
      • Digital pre-distortion
    • Two and more CC cases
  • Transport layer
  • Demod
    • MIMO detection
      • ZF
      • MMSE
      • Sphere-decoding: Tree Search Detection (TSD)
        • QR Decomponsition pre-processing: GS, Modified GS, Householder
        • Traditional K-best search (problems to be solved at each layer): Additive distance metric, Manhattan (L1) norm, addressing missing counter-hypothesis
        • Reduced complexity methodsI, II (again focus on problems to be solved at each layer)
          • Slicing to obtain inherited best candidate
          • Best candidate's sibling
            • 1D Schnorr-Euchner (SE) enumeration (see a picture here or here)
            • 2D search: See figure 2 in this ref
  • Channel estimation
  • Access procedure
    • Synchronization, acquisition and cell search: DL power (AGC), time, freq, beam, cell ID
      • PSS: 5W: SC seq, 3 possible roots, last sym of the first slot of every half frame, (for TDD, it's the 3rd symbol in DwPTS), length 63 SC seq is placed on the 31 REs on each side of the DC. Non-coherent detection using MF
      • SSS: 5W: Interlacing two SS codes, 168, right before PSS, (for TDD, it's 3 symbol before PSS because the first two symbols of DwPTS contains PCFICH, PHICH), length 62 seq is placed on the 31 REs on each side of the DC. Coherent detection using FHT?
      • MIB and PBCH: BW, PHICH info, Sys Frame Num, # of Tx antennas
    • System info (MIB & SIBs)
    • PRACH
      • Freq-time location
      • Generation of RA PA
    • Paging
