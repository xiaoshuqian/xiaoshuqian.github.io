Description

Detailed outline

  • Digital front end, Tx and Rx
    • Single component carrier case: freq down conversion, down-sampling
      • Output sampling rate = nFFT * SCS = normalized BW
      • IQ and DC estimation and compensation
      • AGC
      • Digital pre-distortion
    • Two and more CC cases
  • FFT
    • FFT input rate = nFFT * SCS * 14/15, where 14/15 discounts the CP portion of the output from DFE
    • FFT architectures
      • Pipelined arch
        • SDF = Single-path delay feedback
          • Radix4 SDP (Despain74)
          • Radix22 SDP (Despain74)
        • Others: MDF (Multi-path delay feedback),
      • Single/Dual Mem arch
  • Channel estimation
  • Parameter Est
    • Noise
    • TO
    • Delay spread
    • FO
    • Doppler spread
  • Demapper
    • IRC: Interference Rejection Combining
    • MRC, EGC
    • Sphere detector (ref: chap 5 of Advanced HW Design for ECC)
      • ML detector solves MIMO detection problem optimally for uncoded system
      • For coded system
        • complexity of joint detector and decoder grows exponentially with the codeword length, rendering it impractical
        • Iteravie receiver with a SfISfO (Soft In Soft Out) detector and SISO (Soft In Soft Out) decoder is more practical
        • MAP detector, a SfISfO detector, provides the optimal APP or LLR for the decoder. LSD (List Sphere Detector) provides a candidate list for MAP detector.
        • Jacobian Logarithm: jacln(a,b) = ln(ea + eb) = max(a, b) + ln(1 + exp(-|a - b|))
