Description

Detailed outline

  • Key facts
    • Max 2D symbol rate = One-sided passband BW
    • Spectral efficiency in bits/2D ~ Bits/Hz
    • NSNR = SNR / (2ρ - 1)
    • Symbol Error Rate for QAM = K * Q((3*NSNR)1/2)
    • Symbol Enery or constellation variance for PAM = d2(M2 - 1)/12; Es for QAM = d2(M - 1)/6, where d = distance between nearest constellatin points
    • Channel capacity for AWGN: NSNR > 1 or ρ (bits/Hz) < lg(1+SNR)
    • Need an additional 3dB in SNR for each bit/sym increase. For example, need 6dB increase to go from 16QAM to 64QAM
    • SNR = EbNo * bits_per_symbol. Thus, need 3dB increase in EbNo to go from QPSK to 16QAM and need 9dB increase in EbNo to go from 16QAM to 256QAM.
      • Let EbNo for QPSK be a/2. Then EbNo for 16QAM must be 4a/(2+2) = a and EbNo for 256QAM must be (4**3)a/(2+2*3) = 64a/8 = 8a (i.e 9dB from a)
