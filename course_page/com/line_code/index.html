Description

Detailed outline

  • Line code is often used for wireline transmission when efficiency is desired: power, BW and probability of error. The design emphasis is often on
    • Spectrum matches the channel for power efficiency. Removing DC component in the spectrum is often needed.
    • Presence of clock frequency in the data spectrum for time-synchronization.
    • Some error correction/detection capability to reduce the error rate.
    • Transparency: All sequence can be transmitted
  • Line code construction
    • Bit to symbol mapping:
      Mapping PSD S(ej2πFT)
      Bipolar: A or -A A2
      OOK: 0 or A, =(A + polar)/2 A2/4 (1 + δ(F)/T)
      AMI: 0 or A or -A A2sin2(πFT)
    • Pulse shaping: RZ, NRZ, split-phase
      Pulse p(t) FT P(F)
      NRZ T sinc(πFT)
      RZ T/2 sinc(πFT/2)
      Split-phase T2 sinc2(πFT/2) sin2(πFT/2)
