Description

Detailed outline

  • Introduction
  • Waveform codec
    • DM: Delta Modulation. Key idea: Analysis by Synthesis. Delay in the feedback loop.
    • ADM: Adaptive Delta Modulation. Key idea: Overcome
      • Slope overloading: reconstructed signal cannot increase as fast as the original signal
      • Granular noise: quantized noise. In particular, the resulting quantization error when step size is too big and the signal change is slow.
    • DPCM: Differential PCM. Generalization of DM
    • ADPCM: Generalization of ADPCM and ADM. Key ideas:
      • Predictor output depends both on past reconstructed signal AND current quantized error signal
      • Step size is adaptive.
    • Subband ADPCM: Key idea: Using Quadrature Mirror filters and allocate more bits to signals in the lower halfband
  • Parametric codec
    • Speech production model
    • Linear Predictive Coding (LPC) of speech
    • Codec used in switched network: CELP
    • Codec used in packet network: ROCH
