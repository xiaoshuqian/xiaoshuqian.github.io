AMTH 241, 242, Modern Algebra I & II

Quarter offered

Winter and Spring of 2010

Course description

Introduction to postulational systems; study of integral domains, fields, and rings; special emphasis on group theory and applications. Polya theory and coding theory; Boolean algebra and its relation to switching functions; general theory of lattices, Applications. (2 units)

Description

Detailed outline

  • Groups
    • Definitions
      • Left inverse/identity is also a right inverse/identity
      • Left inverse/identity is unique
      • Permutation law, cancellation law, alternative group definition
    • Subgroups, cosets, Lagrange theorem, isomorphism
    • Examples: unit, symetric, permutation groups
    • Cylic group
      • Examples: modulo, integer
      • Generator, ord(gk) = t/gcd(t,k) if ord(g) = t
      • Existence, structure, quanity problems
      • Polya's theorem
        • Burnside lemma: |M(c)| = |Stab(c)|/|G|, count by row = count by col: Σ |Stab(c)| = Σ |Fix(g)|
        • Three examples:
          • # of ways to seat n people in a table
          • # of ways to paint a baton with n bands using 3 colors: n =2, n = 3, n = even, n = odds
          • # of different necklaces with n beads of 3 colors: n =2, n = 3, n = even, n = odds
      • Normal groups, homomorphism
    • Rings
      • Definitions (cf vector space)
      • Ring with indentity, characteristic of a ring, integral domain, commutative ring, quatient ring
      • Ideals
      • Examples: integers, polynomials
      • Homomorphism
    • Fields
      • Definitions
      • Subfields
      • Examples: finite fields
      • Homomorphism
