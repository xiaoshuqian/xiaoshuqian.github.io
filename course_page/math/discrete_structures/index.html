Description

From ITU course catalog: This course is about discrete structures and forms an introduction to the theoretical side of computer science. Discrete structures and discrete mathematics turn out to be the "calculus" of computer science--these are the structures that students will use to model real-world problems, to build algorithms upon, and to program with (both for modeling problems as well as use in data-structures and algorithms). In this course students will learn about various discrete structures (numbers, sets, relations, functions, trees, graphs), how to talk about them (propositional and predicate logic), how to prove things about them (using contradiction, construction, induction, combinatorics), and how to read and write literate formal mathematics. Students will also get a quick introduction to key applications to algorithmic analysis (like asymptotic worst-case running time analysis for algorithms). This "calculus of computer science" will serve students as a foundation for computational thinking.

Detailed outline

  • Continuous vs discrete calculus
    • Differentiation/Integration vs difference/summation
      • general formulas, examples
    • Differential equation vs difference equation (recurences)
      • Superposition principle
      • Solving homogeneous equation for the general solution
      • Solving inhomogeneous equation for a particular solution
      • Combining the two solutions to satisfy initial condition
      • Master theorem
    • Optional: Euler formula, Bernoulli number
  • Combinatorics
    • Sum rule, Venn diagram
    • Mult rule, tree diagram
    • Arrangement, 4 types: with/without order/repetition
    • Distribution of balls to boxes = arrangement of boxes = sampling
    • Multinomial coefficients
    • Stirling numbers, first and second kind
    • Derangement d(n+1)=n(d(n)+d(n-1))
    • Generating functions
      • Basic examples: GF for binomial/multinomial/negative binomial coefficients
      • GF for Stirling numbers
      • Properties of Fibonacci numbers
        • Discover a formula for F0 + F1 + ... + Fn and prove it
    • Inclusion and exclusion principle
      • Computing 2nd kind Stirling number, derangement, Euler Totient function
    • Pigeonhole principle, Ramsey theorem
    • Summary: Distribution of balls to boxes 12 types: labeled/unlabed ball/box; 3 capacity types
      • pk(n) = # of ways to write n as a sum, with each term smaller than or equal to k = # of ways of placing n identical balls into k or fewer identical boxes (see Daniel Cohen p73, p98 & p128)
  • Propositional logic: syntax, sematics, algebra
  • First order predicate calculus (FOPC)
  • Set theory
    • Set algebra
    • Relation: equivalence, partial/total order
  • Elementary Group theory
    • Polya theorem
  • Number theory
    • Prime numbers
    • GCD, LCM
    • Modulo arithmetic
    • Chinese remainder theorem
    • Fermat-Euler totient theorem, Euler's totient function
    • Mobious inversion
    • Encryption/decryption: classical and public key (RSA & Diffie-Hellman)
      • Cryptography Intro, Online Exercises
  • Graph theory
    • Introductory tutorial from Prof. Chris K. Caldwell
    • Graph, digraph, edge, multiple edges, loop, simple/multigraph, path, cycle, graph representation
