Detailed outline

  • A simple example (see section below on reduction for more examples)
  • Canonical and standard form (using a common convention. No wide agreement on which form is 'standard')
  • Simplex method for LP in cannonical form with b ≥ 0 (Simplex in python: tutorial, packages)
    • To std form: Adding slacking variable to convert the inequality constraints to equality constraints: Ax + Iw = b
    • Write out the initial tableau (target function P - cx = 0), start with the initial non-basic variable x = 0 and then repeat:
      • Pick a pivot column or declare optimal value attained
      • Pick a pivot row or delcare the solution is unbounded
      • Pivot
  • Convert between different forms
  • Phase I solution
    • Common approach
    • Approach described in Vanderbei
  • Degeneracy
    • Different definitions
    • Ways to address degeneracy
  • Duality
    • Dual problem
      • Motivation: estimate the upper bound of the primal problem
      • Alternative motivation: shadow price or marginal cost = maximum price you are willing to pay to get additional resources (if more than that you should be selling instead of producing much less buying more)
    • Weak duality
    • Strong duality
    • Revised simplex method and proof of strong duality
    • Complementary slackness
  • Sensitivity analysis (see Chap 3 slides in Andeson et al. and see this slide set and the AMP book (chap 3) for formal derivation)
    • Sensitivity to the coefficient of the objective function
      • Reduced/opportunity cost of a non-basic variable in the optimal solution = (absolute value of) the dual value on the nonnegativity constraint for that variable = how much the objective function coefficient on the corresponding variable must be improved before the value of the variable will be positive in the optimal solution (reduced cost of a basic variable in the optimal solution is always 0)
      • Range of optimality = The range of values over which an objective function coefficient may vary without causing any change in the values of the decision variables in the optimal solution
        • 100% rule
    • Sensitivity to the limits on the RHS of the constraint inequalities (or equalities)
      • Shadow price or dual value or marginal cost (note that the dual value for a non-negativity constraint associated with a non-basic variable is negative and its absolute value is called reduced cost) = The change in the value of the objective function per unit increase in the right-hand side of a constraint = The shadow price for a constraint is the negative of the coefficient of the slack (or artificial) variable in the objective function of the final tableau (the equivalence of the two expressions can be seen from the final objective function expression in the final tableau as shown in the ref above)
      • Range of feasibility = The range of RHS values over which the dual value is applicable
        • 100% rule
    • Other terms: Sunk cost vs relevant cost = cost independent or dependent on the values of decision variables
  • Reduction and example applications
    • Slides, books etc from Prof. Robert J Vanderbei at Princeton
      • Portforlio selection
      • Option pricing
      • Network type problems
