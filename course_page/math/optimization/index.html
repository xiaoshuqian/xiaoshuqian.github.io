Detailed outline

  • LS
    • Regularized LS: How to choose λ?
  • Gradient descent: repeat {determine search dir; line search to get new point}
    • Steepest descent: -g
    • Newton method: -H-1g
    • Varaible metric methods (Quasi-Newton): -Bg, B ~ H-1: generalization of the secant method to find the root of the first derivative for multidimensional problems.
      • See popular update formulas: BFGS, Broyden, DFP, SR1
    • Conjugate gradient: s' = -g + βs, where β = r'Tr' / (rTr)
      • Residuals are orthogonals to each other and are orthogonals to all previous search directions
      • Search directions are conjugate-orthogonals to each other
      • Preconditioning is typically used when using CG for large matrix
  • Direct search methods:
    • Down-hill simplex (Nelder-Mead)
    • Conjugate direction
  • Constrained optimization methods
