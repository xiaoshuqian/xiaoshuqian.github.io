Detailed outline

  • Set algebra
  • Relations
    • A pre-order or quasi-order is transitive and reflexive
    • An equivalence relation is a symmetric pre-order
    • A (non-strict) partial order is an anti-symmetric pre-order (≤)
      • Anti-symmetric: aRb and bRa implies a = b
      • Poset: a partially ordered set
      • Lattice: a partially ordered set where every two elements has a least upper bound (joint ∨) and greatest lower bound (meet ∧)
        • meet (joint) defined above is commutative, associative and idempotent
        • Conversely, we can define meet as a commutative, associative and idempotent relation and use it to define a partial order: a ≤ b iff a ∧ b = a
      • A strict partial order can be defined by partial order: a < b is defined by: a ≤ b and a ≠ b
    • A (non-strict) total order is a partial order (≤) that exists for every pair of the set
      • Trichotomous: for every pair (a, b) exactly one of aRb, bRa and a = b is true
      • A strict total order can be defined by total order: a < b is defined by: a ≤ b and a ≠ b
